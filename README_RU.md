# Filato CAM PRO Android V9 Adaptive UI

Тестовая Android-ветка для сборки APK через GitHub Actions.

Что изменено в V9:
- убрана принудительная альбомная ориентация;
- приложение работает в книжной и альбомной ориентации;
- интерфейс перестроен вертикальными блоками без горизонтальной прокрутки;
- уменьшены размеры элементов, чтобы ничего не срезалось по X;
- микрофон и системная диктовка временно отключены;
- оставлен текстовый разбор заказа;
- блоки идут сверху вниз: параметры станка, заказ/расчет, 2D preview, G-code, проверка/экспорт;
- экспорт ZIP создает внутри архива папку с именем заказа: CNC, SUMMARY, .filato-order;
- есть отправка текущего CNC через системное меню "Поделиться".

Важно: CNC-геометрия пока тестовая. На станке не использовать до переноса ArtCAM-пресетов.
